代码随想录 - 数组

​ 按照link进行刷题学习，具体信息请点击代码随想录官网。

1、数组的理论基础

​ 数组是存放在连续内存空间上的相同类型的数据的集合。数组的下标都是从0开始的，内存空间的地址是连续的。

2、二分查找-704

​ 在有序数组中进行目标值的查找，可使用二分查找。该算法非常重要，在一系列相似题目中有各种各样的变形，因此对应的模板是要烂熟于心的，并且清楚每一部分的对应含义。

​ 二分查找的模板有两种，我一般习惯使用第一种。不同模板的语句对应的含义最好要弄懂，对后续二分查找的变形题目会有帮助。

​ 以上两种模板的主题思想是一样的，就是不断比较中间位置的值与目标值，然后每次缩减一半的搜索空间。核心的两点不同在于while循环的条件和right的更新值。因为我个人是喜欢用模板1的，因此后续代码我都会用模板1 写。

3、二分查找-例题1（搜索插入位置）-35

​ 题目：给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

​ 这道题与传统二分查找问题唯一的不同在于，返回的是索引值或被插入的位置。仔细思考二分查找左右指针的移动规律，就可以得到这道题的答案。

4、二分查找-例题2（在排序数组中查找元素的第一个和最后一个位置）-34

​ 题目：给定一个按照升序排列的整数数组 nums，和一个目标值 target。找出给定目标值在数组中的开始位置和结束位置。如果数据中不存在目标值，则返回[-1, -1]。

​ 这个题的含义就是，找到连续目标值的左边界和右边界。理解这道题对深入理解二分法非常有帮助。

5、二分查找-例题3（x的平方根）-60

​ 题目：给你一个非负整数x，计算并返回x的算数平方根。由于返回类型是整数，结果只保留整数部分，小时部分将被舍去。

​ 可以理解为，寻找最大的平方小于x的整数，可以使用二分法进行查找。

6、二分查找-例题4（有效的完全平方数）-367

​ 题目：给定一个正整数num,编写一个函数，如果num是一个完全平方数，则返回True，否则返回False。

​ 可以借助二分查找法，去看有没有一个数的平方是目标正整数，然后不断更新左右边界。

7、移除元素-27

​ 题目：给你一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素，并返回移除后数组的新长度。不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并 原地 修改输入数组。

​ 使用快慢指针的方法，在原地实现移除元素的功能。

8、移除元素-例题1（删除排序数组中的重复项）-26

​ 题目：给你一个 升序排列 的数组 nums ，请你** 原地** 删除重复出现的元素，使每个元素 只出现一次 ，返回删除后数组的新长度。元素的 相对顺序 应该保持 一致。

​ 和7一样，只是多存储一个变量用来标记之前元素的值，用来判断是否是连续重复的元素，从而进一步来删除。

9、移除元素-例题2（移动0）-283

​ 题目：给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。请注意 ，必须在不复制数组的情况下原地对数组进行操作。

​ 一毛一样的用快慢指针，只是记得把剩下的数变为零。

10、移除元素-例题3（比较含退格的字符串）-844

​ 题目：给定 s 和 t 两个字符串，当它们分别被输入到空白的文本编辑器后，如果两者相等，返回 true 。# 代表退格字符。注意：如果对空文本输入退格字符，文本继续为空。

​ 实际上就是每碰到退格就对之前的字符串做一个移除元素处理。需要注意这个注意，注意慢指针不要到-1，我在提交的时候被这个坑过。

11、有序数组的平方 - 977

​ 题目：给你一个按 非递减顺序 排序的整数数组 nums，返回 每个数字的平方 组成的新数组，要求也按 非递减顺序 排序。

​ 无非就是从有序数组里面找出按绝对值排序的序列。

12、长度最小的子数组-209

​ 题目：给定一个含有 n 个正整数的数组和一个正整数 target。找出该数组中满足其和>=target的长度最小的连续子数组，并返回其长度。如果不存在符合条件的子数组，返回0.

​ 挑取连续子数组，应该第一反应想到的是滑动窗口的算法。滑动窗口的起始位置应该是所有求和小于target的位置，终点位置应该是遍历整个数组。

13、螺旋矩阵2-59

​ 题目：给你一个正整数 n ，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。

​ 这道题的重点不在于相关算法，而在于如何用代码模拟整个过程。这一模拟的过程对代码的细节问题要求较高。主要就是方向和细节上的问题。
