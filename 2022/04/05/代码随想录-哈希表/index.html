代码随想录 - 哈希表

1、哈希表理论基础

​ 哈希表是根据关键码的值而直接进行访问的数据结构。

2、有效的字母异位词-242

​ 题目：给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。

​ 就是统计两个字符串中字符出现的频率是否一样。好像也可以直接对字符串排序然后看两个字符串是否一样，但是复杂度高了。这里对两个字符串进行字符的统计其实使用的就是哈希表。

3、两个数组的交集-349

​ 题目：给定两个数组 nums1 和 nums2 ，返回 它们的交集 。输出结果中的每个元素一定是 唯一 的。我们可以 不考虑输出结果的顺序。

​ 这个简单就不写了。

5、两数之和-1

​ 题目：给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。

​ 最简单的就是暴力解法，但是有n^2的复杂度。用哈希表则可以做到用空间换时间，做到n的复杂度。通过一个哈希表，建立值到索引的对应关系，通过判断需要的值是否在哈希表中来判断二者的和是否为目标值。

7、赎金信-383

​ 题目：给你两个字符串：ransomNote 和 magazine ，判断 ransomNote 能不能由 magazine 里面的字符构成。

​ 看起来就是判断一个字符串是不是另一个字符串打乱后的子集的题。可以使用哈希表来记录已经有的字符的个数。

8、三数之和-15

​ 题目：给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有和为 0 且不重复的三元组。

​ 这道题不仅需要找出三个数使其和为零，而且还要做去重的操作，挺麻烦的。

9、四数之和-18

​ 题目：给你一个由 n 个整数组成的数组 nums ，和一个目标值 target 。请你找出并返回满足下述全部条件且不重复的四元组 。

​ 这道题跟三数之和用一个方法做就可以了，就是在外面多套一层循环。
